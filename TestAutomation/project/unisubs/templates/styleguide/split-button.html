Split buttons are useful if you want to pack a primary action and a set of secondary actions in a single element. For example, transcription as the primary action and translation to other languages as the secondary actions.

The code for split buttons is at ui.utils.SplitCTA. The most important parameters for SplitCTA's constructor are `menu_id` and `dropdown_items`. Make sure that each SplitCTA has a unique `menu_id` whenever there are multiple of them in a single page.
